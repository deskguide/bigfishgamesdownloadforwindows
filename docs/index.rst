Common Download Problems
========================

Download Won't Start
--------------------

If the download does not begin:

* Check your internet connection.
* Refresh the download page.
* Restart your web browser.
* Verify that you have enough available storage.

Installer Won't Open
--------------------

If the installer doesn't launch:

* Download the installer again.
* Right-click the installer and choose **Run as administrator** if needed.
* Restart your computer and try again.

Installation Failed
-------------------

If installation stops unexpectedly:

* Close unnecessary applications.
* Ensure Windows 10 is up to date.
* Verify that your computer meets the game's minimum requirements.
* Download a fresh copy of the installer.

Game Won't Launch
-----------------

If the game won't open after installation:

* Restart your PC.
* Check for Windows updates.
* Install available game updates.
* Reinstall the game if the issue continues.

Do I need a Big Fish Games account?
-----------------------------------

Yes. An account is typically required to access purchased games, manage your game library, and download eligible titles.

Can I install games on multiple Windows 10 computers?

Many purchased games can be installed on multiple compatible computers using the same account, subject to the applicable license terms.

Why won't my game install on Windows 10?
----------------------------------------

Installation issues may result from insufficient storage, outdated Windows software, incompatible system requirements, or a corrupted installer. Updating Windows and downloading the installer again often resolves these problems.

How do I update my installed games?
-----------------------------------

Game updates are generally provided through the game's launcher or the platform used to install the game. Keeping both Windows 10 and your games up to date helps ensure the best performance.
